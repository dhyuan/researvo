# Basic Survey Type Coverage

## Description

This survey is a positive test fixture for the creation wizard Markdown import flow. It covers the main supported survey question types in one concise respondent experience.

## Goal

Verify that the wizard can import a Markdown brief and guide schema generation for single choice, multiple choice, short text, long text, numeric, Likert scale, and yes/no style questions.

## Questions

1. What is your primary role when using this product?
   - Student
   - Researcher
   - Product manager
   - Designer
   - Engineer
   - Other

2. Which product activities have you done in the past month? Select all that apply.
   - Created a survey
   - Edited survey questions
   - Previewed a survey
   - Published a survey
   - Reviewed responses
   - Exported results

3. In a few words, what name would you give this workflow?

4. Please describe one improvement that would make this product more useful for you.

5. How many surveys have you created in the past 30 days?

6. How satisfied are you with the current survey creation experience? Use a 1 to 5 scale where 1 means very dissatisfied, 3 means neutral, and 5 means very satisfied.

7. Have you published at least one survey before?
   - Yes
   - No

## Constraints

Do not collect names, phone numbers, email addresses, street addresses, account IDs, or other directly identifying information. Keep all questions optional unless the respondent must consent before continuing.
